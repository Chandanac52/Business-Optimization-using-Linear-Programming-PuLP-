# Business Optimization using Linear Programming (PuLP)

## Project Overview

This project focuses on solving a real-world business optimization problem using Linear Programming. The objective is to maximize profit by determining the optimal production mix of two products (Product A and Product B) while considering constraints such as labor and material availability.

The model is implemented in Python using the PuLP library and demonstrates how mathematical optimization techniques can be applied to improve decision-making in business scenarios.
